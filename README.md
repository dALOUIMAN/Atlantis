## GitHub Enterprise Cloud Trial Setup

This project uses GitHub Enterprise Cloud. The following outlines what is included in the trial and how to get started.

### What is included in the trial?

The trial lasts for **30 days** and includes:

- Access to most GitHub Enterprise Cloud features
- GitHub Copilot Business
- GitHub Secret Protection and GitHub Code Security
- An enterprise account (allows managing multiple organizations)
- Up to **50 licenses** to grant access to users

### Features not included in the trial

- GitHub Codespaces
- GitHub Copilot Enterprise
- GitHub Sponsors
- Paid GitHub Marketplace apps
- GitHub Connect
- Git Large File Storage
- For GitHub Actions: increased minutes, job concurrency, and larger runners

### Payment

No payment method is required to start a trial. If you want to use Copilot Business during the trial, a credit card is needed — but you **will not be charged** during the trial period.

### During the trial

- You can create up to **three new organizations** in the trial enterprise, or transfer any number of existing organizations.
- Organizations created during the trial cannot be removed from the enterprise account until you purchase GitHub Enterprise.
- Billing for transferred organizations is paused during the trial and any coupons are removed.
